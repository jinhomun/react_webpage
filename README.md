# 리액트를 이용한 웹 사이트 만들기
처음으로 리액트 사이트를 만들어 봅니다.

리액트란,싱글 페이지 애플리케이션(SPA)과 웹 애플리케이션의 프론트엔드 개발에 주로 사용되며, 여러 웹 애플리케이션에서 UI 구축을 단순화하고 향상시키는 데 도움을 줍니다.


## 설치
npx create-react-app react_webpage
